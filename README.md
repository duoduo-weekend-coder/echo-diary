# Echo Diary

A warm, minimal personal diary app built with React + Vite + Tailwind CSS.

## Features

- **Daily Entries** — Write text, record voice notes (MediaRecorder API), and attach photos
- **Timeline View** — Chronological feed of all entries, grouped by date
- **Week View** — 7-day grid with per-day entry counts and quick navigation
- **Month View** — Calendar with entry indicators; tap any day to see its entries
- **Year View** — 12-month heatmap showing your writing frequency at a glance
- **On This Day** — The same calendar date across all years, stacked vertically like an echo
- **Local Storage** — All data lives in the browser, completely private, no server needed

## Tech Stack

- React 18
- Vite 6
- Tailwind CSS 3
- Lucide React (icons)
- Google Fonts: Fraunces + Lora + DM Sans

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

npm run build
